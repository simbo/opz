The "OpZ" application is not signed with an Apple developer certificate.

After downloading, macOS may block the app and display a warning.

To run OpZ:

1. Open System Preferences.

2. Navigate to "Security & Privacy" > "General".

3. Click "Open Anyway" next to the OpZ warning.

4. Confirm in the dialog.

Repeat these steps if the issue returns after updating OpZ to a new version.
